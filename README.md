# ng2-start

This is a minimal setup to start creating an angular2 application with webpack.

### Instructions

Requirements

```json
{
  "node": ">=6"
}
```

Setup environment

```bash
# cd into ng2-start
cd ng2-start

# Install dependencies
npm install
```

Serve and Build
```bash
# Start dev server at http://localhost:9000
npm start

# Build
npm run build

# Build for production (this will also use aot compiler)
npm run build:prod
```
